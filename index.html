Тег table является родителем для тегов tr. Каждый тег tr является родителем для тегов td. Теги td являются родителями для текста, который находится внутри них.

Потомками абзаца являются теги i, b, a и i. Непосредственными потомками абзаца являются теги i и a.

Родителями тега b являются теги i, a и p. Непосредственным родителем тега b является тег i.
